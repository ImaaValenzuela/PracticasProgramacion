# #06 Clases
> #### Dificultad: Fácil 

## Ejercicio

```
1️⃣ Creación de Clases y Objetos
📌 Ejercicio:
Crea una clase Coche con los atributos marca, modelo y velocidad.
✔ Agrega un constructor para inicializar los valores.
✔ Agrega un método acelerar() que aumente la velocidad.
✔ Agrega un método frenar() que disminuya la velocidad.
✔ En main(), crea un objeto, llama a los métodos y muestra la información.

2️⃣ Uso de new y null
📌 Ejercicio:
✔ Declara una variable de tipo Coche y asigna null.
✔ Luego, intenta llamar a un método y observa el error.
✔ Corrige el error verificando si la variable es null antes de llamar al método.

3️⃣ Garbage Collector
📌 Ejercicio:
✔ Crea una clase Persona con un mensaje en el constructor ("Persona creada").
✔ En main(), crea varios objetos Persona, pero no los almacenes en una variable.
✔ Observa que el GC eliminará algunos objetos sin referencia.

4️⃣ this y Ocultamiento de Atributos
📌 Ejercicio:
✔ Modifica la clase Coche y usa this en su constructor para evitar ocultamiento de atributos.

5️⃣ Métodos y Atributos Estáticos
📌 Ejercicio:
✔ Crea una clase Calculadora con un método estático sumar(int a, int b).
✔ Llama al método en main() sin instanciar la clase.

6️⃣ Paquetes
📌 Ejercicio:
✔ Crea un paquete vehiculos y dentro una clase Moto.
✔ En otra clase Main, importa vehiculos.Moto y usa un objeto de la clase.

7️⃣ Modificadores de Acceso
📌 Ejercicio:
✔ Crea una clase CuentaBancaria con private double saldo.
✔ Agrega getSaldo() y depositar(double monto).
✔ En main(), crea un objeto CuentaBancaria y usa sus métodos.

8️⃣ Encapsulamiento con Get y Set
📌 Ejercicio:
✔ Agrega setSaldo(double saldo) en CuentaBancaria, validando que no sea negativo.
```
