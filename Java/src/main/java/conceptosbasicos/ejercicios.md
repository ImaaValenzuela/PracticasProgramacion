# #01 Variables, constantes, operadores y expresiones.
> #### Dificultad: Fácil 

## Ejercicio

```
1. Identificadores
Crea variables con nombres válidos e inválidos en Java y explica por qué algunos nombres no son permitidos.

2. Tipos de Datos Primitivos
Declara variables de diferentes tipos primitivos y muestra sus valores.

3. Palabras Reservadas
Investiga y enumera 10 palabras reservadas en Java. ¿Por qué no se pueden usar como identificadores?
abstract, assert, boolean, break, byte, case, catch, char, class, continue
No se pueden usar como identificadores porque son palabras reservadas por el lenguaje que tienen un uso especifico.

4. Declaración de Variables y Tipos
Declara una variable local dentro de un método y una variable estática en una clase.

5. Constantes y Variables
Declara una constante en Java con final.

6. Operadores
Escribe una expresión que utilice diferentes tipos de operadores y muestra el resultado.

7. Expresiones
Escribe una expresión matemática compleja y evalúa su resultado.

8. Comentarios
Agrega comentarios explicativos en un código sencillo.

9. Conversión de Tipos
Convierte un tipo de dato a otro y muestra el resultado.

10. Tipos de Datos Objeto
Crea un objeto String y realiza algunas operaciones con él.

```
