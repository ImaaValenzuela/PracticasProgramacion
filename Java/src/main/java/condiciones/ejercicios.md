# #03 Estructuras de Control: Selección
> #### Dificultad: Fácil 

## Ejercicio

```
1. Condicional Simple (if)
Determina si un número ingresado por el usuario es positivo.

2. Condicional Doble (if…else)
Verifica si un número ingresado es par o impar.

3. Anidación de Condicionales
Evalúa si un número es positivo, negativo o cero.

4. Operador Ternario (?)
Determina si una persona es mayor de edad.

5. Condicional Múltiple (switch)
Convierte un número del 1 al 7 en el nombre del día de la semana.

6. Anidación de Condicionales Múltiples
Evalúa el rendimiento de un estudiante en función de su calificación.

```
