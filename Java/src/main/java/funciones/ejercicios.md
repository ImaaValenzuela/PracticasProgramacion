# #05 Funciones
> #### Dificultad: Fácil 

## Ejercicio

```
1. Programación Modular (Uso de Funciones)
Crea una función que sume dos números y retorne el resultado.

2. Ámbito de Variables (Global vs Local)
Ejemplo donde una variable global puede ser accedida en cualquier función, pero una local solo dentro de su método.

3. Pasaje de Parámetros
Distinguir entre paso por valor (primitivos) y paso por referencia (objetos).

4. Retorno de Valores con return
Función que retorna el mayor de dos números.

5. Sobrecarga de Métodos
Definir funciones con el mismo nombre pero diferentes parámetros.

6. Recursividad
Calcular el factorial de un número usando recursión.

7. Declaración y Definición de un Método
Función que verifica si un número es primo.

8. Argumentos de main
Ejecutar un programa pasando argumentos por línea de comandos.
```
